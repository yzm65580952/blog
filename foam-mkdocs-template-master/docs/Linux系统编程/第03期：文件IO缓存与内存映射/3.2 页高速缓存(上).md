# 3.2 页高速缓存(上)
## 页缓存
通过Linux内核缓冲区实现了一种磁盘缓存机制
基本原理：局部原理、时间局部、空间局部
缓存基于页的对象：普通文件、块设备、内存映射
优点：一定程度上分离了应用程序空间和物理设备、减少IO读盘次数

## 同步方式
1. fsync(fd)同步
2. sync 刷新所有的页
3. open(filename,O_SYNC)同步写


## 页缓存读写实验
```c

dd if=/dev/zero of=test.dat bs=1M count=10
// 往test.dat写0 写10次 每次1M
cat /proc/meminfo | grep Dirty

sync

cat /proc/meminfo | grep Dirty
```
