# 8.4 获取当前的时间

 ## 全局变量：xtime
 ```c
typedef long time_t;
time_t time(time_t *tloc);
int stime(const time_t *t);
```

## demo 
```c
#include <stdio.h>
#include <time.h>

int main (void)
{
    time_t t = time (NULL);	//获取当前时间
    printf ("%ld\n", t);
    t = t -100;
    stime (&t);			//设置时间
    t = time (NULL);
    printf ("%ld\n", t);
    return 0;
}

```
